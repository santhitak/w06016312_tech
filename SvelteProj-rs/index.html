SVELTE คืออะไร ?

เป็นฟรอนต์เอนด์เฟรมเวิร์ค แบบหนึ่ง เช่นเดียวกับ Angular, React และ Vue แต่ Svelte เป็น เฟรมเวิร์ค ที่ไม่ใช่ เฟรมเวิร์ค คือ Angular, React และ Vue จริงๆเป็นแค่จาวาสคริปต์ไลบรารี่ ที่เวลาจะใช้งาน ก็ download และ import เข้ามา และเรียกใช้ module ต่างๆใน library แต่ Svelte นั้นต่างกัน Svelte เป็นภาษาใหม่ โดยมี Compiler ในการแปลงภาษานั้นให้เป็น Javascript ภาษา Svelte นั้นเป็น Javascript ที่ถูกดัดแปลง Syntax เล็กน้อย เพื่อให้เหมาะกับการเขียนเว็บไซด์ เมื่อเขียนเสร็จก็ compile .svelte ไปเป็น Javascript ปกติ และก็สามารถแสดงผลเป็นเว็บไซด์ธรรมดาได้เลย

” I can confidently say that Svelte has been the easiest JavaScript component library to learn and start putting to use in a productive way “

Jeff Delaney, from Svelte Realtime Todo List with Firebase


อะไรที่ทำให้ Svelte แตกต่าง ?

ยืดหยุ่นและเขียนน้อยกว่า

Svelte เป็น Compiler นั่นหมายความว่า code ในไฟล์ Svelte จะถูกแปลงจาก Hybrid language ที่ผสมระหว่าง HTML, CSS, JavaScript ให้เป็น lower-level optimize file แทน ทำให้การเขียน code มีความยืดหยุ่นมากขึ้น รวมทั้งยังเป็นประโยชน์ต่อเว็บเบราว์เซอร์อีกด้วย

No virtual DOM

DOM (Document Object Model) คือ interface ที่กำหนดโครงสร้างของเว็บไซต์ มันจะนำ HTML มาเปลี่ยนเป็นโครงสร้างที่สามารถจัดการและเข้าถึงได้ Svelte มีแนวทางที่แตกต่างออกไป นั่นคือการไม่ใช้ Virtual DOM นั่นทำให้แอพพลิเคชั่นมีประสิทธิภาพที่ดีขึ้น

CSS scope

Svelte จะนำ CSS จากแต่ละ component มาแบ่งให้เป็น CSS แต่ละไฟล์ในช่วงระหว่าง build ยกตัวอย่างเช่นใน React จะเป็นการจัดเรียงแต่ละ component ทั้ง markup และ JavaScript ส่วน Svelte จะเป็นการรวบรวมแทนทั้ง markup และ JavaScript ให้อยู่ใน”.svelte” ไฟล์เดียว

เปรียบเทียบความยาวสำหรับ code แบบเดียวกันใน react และ svelte
เปรียบเทียบ complier และ dom
โครงสร้างของ svelte

ข้อดี

  • การห่อหุ้มสไตล์ในตัว
  • การคอมไพล์โค้ดในเวลาสร้าง
  • ความเร็วที่เร็วขึ้นและประสิทธิภาพที่ยอดเยี่ยม
  • มีขนาดกลุ่มแอปพลิเคชันขนาดเล็ก
  • ไม่มี DOM เสมือน
  • CSS-in-JS Library

ข้อเสีย

  • ช้าในการลบองค์ประกอบ
  • ไม่มี DevTools เพื่อดีบักแอปพลิเคชันหลังการคอมไพล์
  • Library ยังไม่ค่อยสมบูรณ์

Tips!  Svelte เมื่อเทียบกับ เฟรมเวิร์ค อย่าง Angular, React และ Vue ก็คือ ไฟล์เมื่อ Compile ออกมาแล้วมีขนาดเล็กและเร็วมาก ที่เป็นเช่นนั้นเพราะ ทุกตัว เป็น Library และ Virtual DOM เวลาใช้งานจะต้องดาวน์โหลดโค้ดทั้งหมดมาใส่ไว้ในเว็บ ทำให้ประสิทธิภาพช้ามาก แต่ Svelte แค่เปลี่ยนโค้ดจากภาษา svelte ไปเป็น Javascript โดยกระบวนการระหว่างเปลี่ยนโค้ด ได้มีการ Optimize ให้โค้ดเหลือสั้นกระชับที่สุด ทำให้โค้ดที่เอาไปใช้งานจริงเล็กมาก และเร็วมาก

เริ่มต้นใช้งาน Svelte !

1. ใช้ the REPL

cd /path/to/svelte-app
npm install

npm run dev

สิ่งนี้จะให้ทำการ run บน localhost:5000 และ roll-up ทุกครั้งที่คุณทำการเปลี่ยนแปลงไฟล์ใน svelte-app/src

2. ใช้ degit

npx degit sveltejs/template my-svelte-project
cd my-svelte-project
# to use TypeScript run:
# node scripts/setupTypeScript.js
npm install
npm run dev

สิ่งนี้จะให้ทำการ run บน localhost:5000 ทำการติดตั้ง dependencies และ roll-up ทุกครั้งที่คุณทำการเปลี่ยนแปลงไฟล์ใน svelte-app/src

npx degit your-name/template my-new-project

ใช้คำสั่ง npm run build เพื่อสร้างเวอร์ชั่นที่พร้อมใช้งาน
